PDF storage for public links.

How to use:
1. Copy your PDF files into this folder.
2. Keep file names URL-safe (lowercase, hyphens, no spaces), for example:
   `errori-comuni.pdf`
3. PDFs in this folder are publicly available at:
   `/pdfs/<file-name>.pdf`
